# v1.0.0-rc.1 Release Notes

Status: release candidate documentation ready.

Important boundary: `v1.0.0` tag creation and GitHub Release publication are
human-only final actions. The toolchain must not create the final tag or release
without explicit human approval.

## Summary

`Claude-StartUpTools-New-Windows` is the Windows-native successor of the
upstream `ClaudeCode-StartUpTools-New` runtime. The active operating model is
PowerShell, Node.js, Windows Task Scheduler, D-drive Project Registry,
Supervisor, and Mission Control.

## Included In This RC

| Area | Included capability |
|---|---|
| Windows root | New Git root and GitHub repository |
| Legacy isolation | SSH/Linux/tmux/bash runtime assets isolated under `legacy-linux/` |
| Project Registry | D-drive scan, registration, GitHub remote detection, schema checks, backup, restore |
| Supervisor | Registered-project autonomy with concurrency, cooldown, restart limits, and stop reasons |
| Mission Control | Windows labels, Registry summary, Supervisor status, GitHub links, AutoRun state |
| CI | Windows runner, Pester unit gate, Node smoke, PSScriptAnalyzer, manual dispatch |
| Docs | README, Windows operations, migration audit, RC checklist, release gate docs |
